# HR-analytics-using-powerbi
1. Project Title / Headline -
  Analyzed employee data to identify attrition, satisfaction, and workforce trends, helping HR make data-driven decisions to improve retention and performance.

2. Short Description / Purpose -
   To analyze employee data and help HR make better decisions on retention, performance, and workforce planning.

3. Tech Stack -
   * Power BI Desktop – Main data visualization platform used for report creation and 
     interactive dashboard design.
   * Power Query – Data transformation and Data Cleaning .
   * DAX (Data Analysis Expressions) – Used for calculated measures, key performance indicators (KPIs).
   * File Format – .pbix for development and .png for dashboard previews and exports.

4. Data Source - I did this project as part of my data analytics course.

5.  Features / Highlights - GOAL OF THE DASHBOARD - KPI Cards - Snapshot metrics for quick decision-making (employee count, attrition count, attrition rate %, salary). Department Filter  - Sliceable dashboard , allows drill-down by HR, R&D, Sales. Mixed Visualization Types - Bars (categorical), area chart (trends over time), table (detailed breakdown). 

Walkthrough of Key Visuals - 
* Visual #1: Attrition Rate by AgeGroup (Bar Chart) - Youngest (18-25): 36 % attrition rate. Oldest (55+): 9 % attrition rate. Pattern: Clear inverse relationship. Junior talent leaves 4x faster than seniors.
* Visual #2: Attrition Rate by SalarySlab (Bar Chart).Upto 5k: 0.22 (highest). 15k+: 0.04 (lowest). Lower-paid employees leave more frequently.
* Visual #3: Attrition Count by Years at Company (Area Chart). Peak at Year 2 — Largest attrition spike early in tenure .Drops significantly by Year 3-4. Becomes stable/predictable after Year 5
* Visual #4: Attrition Rate by Job Role (Bar Chart). Sales Representative: 0.40 ⚠️ Critical risk. Laboratory Technician: 0.20. HR: 0.20 .Sales Executive: 0.20. Research Scientist: 0.20

 Insights - HR — Young staff bailing at 50%. No middle positions. They see nowhere to go, so they leave.
            R&D — Nailing it. Lowest pay but best retention. People stay because work is interesting, not just the paycheck.
            Sales — Disaster. Paying the most but losing 40% of reps in Year 1. Quotas are probably too harsh for new hires.
